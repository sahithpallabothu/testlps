LPS requires the following browser settings to be set before proceeding:

1. Open Internet Explorer.
2. Click on Tools menu and select Internet Options.
3. Click on the Security tab, then click on Internet zone.
4. Click on Custom level button and scroll to User Authentication settings.
5. Select "Automatic logon with current username and password" option and click on OK.
6. Click on Apply in Security tab to apply the selected Option.
7. Close Internet Explorer and you are now set to use the application.

1. Open Firefox, type "about:config" into the address bar and hit enter.
2. Firefox will warn you that incorrectly editing your settings may lead to stability, security and performance issues. Click the "I accept the risk!" button.
3. Key in "network.automatic-ntlm-auth.trusted-uris" in the search bar that shows up at the top. As you start typing in, the list that is shown below gets filtered.
    Right-click the setting and select "Modify".
4. The resulting popup lists websites that Firefox will automatically log into.
    Add the URL of the server that hosts the application (ex:{{serverIp}} ) to have Firefox automatically log in as the currently logged on Windows user.
5. If there already exists another site, you can key in a comma towards the end of the existing value, followed by the URL of the server.
6. Restart Firefox.
